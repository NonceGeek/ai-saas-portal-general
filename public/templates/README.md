# 数据标注模板说明

## 表头定义

| 表头 | 必填 | 数据格式 | 说明 | 示例 |
|------|------|----------|------|------|
| 字符 | ✓ | 单个汉字或词组 | 要标注的粤语字符 | 行 |
| 分类 | ✓ | 字符串 | 数据分类，默认为 zyzdv2 | zyzdv2 |
| 粤音 | ✓ | 用 \| 分隔多个粤音 | 该字符的粤语读音 | hang4\|haang4\|hong4 |
| 组词 | × | 用 \| 分隔多个组词 | 包含该字符的词组及释义 | 步～\|～路\|發～ |
| 句子 | × | 用 \| 分隔多个句子 | 包含该字符的例句 | 佢行得好快\|今日做乜行？ |
| 相关文献 | × | 格式：名称->链接，用 \| 分隔多个 | 相关的文献资料 | 粤语词汇研究->https://example.com/doc1 |
| 视频切片 | × | 格式：名称->链接，用 \| 分隔多个 | 相关的视频资料 | 粤语教学视频->https://example.com/video1 |

## 数据格式规范

### 1. 字符 (必填)
- 单个汉字或多个字符组成的词组
- 不能为空

### 2. 分类 (必填)
- 默认值：zyzdv2
- 用于区分不同类型的语料

### 3. 粤音 (必填)
- 使用粤语拼音标注
- 多个读音用 `|` 分隔
- 示例：`hang4|haang4|hong4`

### 4. 组词 (可选)
- 包含该字符的词组和释义
- 多个组词用 `|` 分隔
- 示例：`㈠①走：步～．～路。②流通。③傳遞：發～。④能幹：他真～|㈡①排：一目十～。②職業：同～。③商店：車～．～市`

### 5. 句子 (可选)
- 包含该字符的例句
- 多个句子用 `|` 分隔
- 示例：`佢行得好快|今日做乜行？|呢行生意唔好做`

### 6. 相关文献 (可选)
- 格式：`文献名称->文献链接`
- 多个文献用 `|` 分隔
- 示例：`粤语词汇研究->https://example.com/doc1|广东话字典->https://example.com/doc2`

### 7. 视频切片 (可选)
- 格式：`视频名称->视频链接`
- 多个视频用 `|` 分隔
- 示例：`粤语教学视频1->https://example.com/video1|粤语教学视频2->https://example.com/video2`

## 注意事项

1. **必填字段**：字符、分类、粤音 三个字段必须填写
2. **分隔符**：使用英文竖线 `|` 分隔多个值
3. **链接格式**：相关文献和视频切片使用 `->` 连接名称和链接
4. **编码**：请确保文件保存为 UTF-8 编码
5. **文件格式**：支持 .xlsx 和 .xls 格式

## 批量上传流程

1. 下载模板文件
2. 按照格式填写数据
3. 保存文件
4. 在系统中点击"批量上传"
5. 选择填写好的文件
6. 系统会先验证数据格式
7. 验证通过后确认上传
